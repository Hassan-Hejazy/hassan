CONNECTED CINEMATIC V14 — VALIDATION REPORT

Core corrections
- Camera fitting now uses the actual service architecture bounds only. Oversized platforms, rings, route graphics and labels are excluded from zoom calculations.
- Corrected local/world coordinate handling that previously doubled station positions and produced inaccurate camera targets.
- Corrected desktop composition centering so each model uses the full available visual frame beside the copy panel.
- Mobile portrait scenes use a dedicated centered safe frame above the information panel.
- Transitions use a continuous route-following camera flight. Full environments are no longer shrunk to nearly zero during service changes.
- Camera movement uses a critically damped scroll spring with no delayed capped-step behavior.
- Station camera poses are cached, reducing connected-scene initialization cost while preserving smooth interpolation.
- Adaptive pixel density and frame-time monitoring preserve sharp output without forcing an unstable literal 4K framebuffer on phones.
